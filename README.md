# Credit Card Fraud Detection

## Overview

This project uses Machine Learning to detect fraudulent credit card transactions.

## Dataset

Kaggle Credit Card Transactions Fraud Detection Dataset

## Algorithms Used

1. Logistic Regression
2. Decision Tree
3. Random Forest

## Results

| Model               | Accuracy|
|---------            |-------- |
| Logistic Regression | 95.36%  |
| Decision Tree       | 99.45%  |
| Random Forest       | 99.57%  |

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Features

- Data Preprocessing
- Fraud Detection
- Model Comparison
- Saved ML Model
- Transaction Prediction
